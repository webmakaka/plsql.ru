### Ограничение числа выбираемых данных (WHERE)


С помощью WHERE  можно ограничить число выводимых оператором SELECT строк согласно определенного критерия.

<br/>

    SELECT name AS "ФИО", matches AS "Матчи", goals "Голы"
    FROM russian_team
    WHERE matches > 50;
